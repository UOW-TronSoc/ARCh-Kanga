# kanga_cameras

Kanga integration for ROS-compatible cameras.

## Owns

- Kanga-specific camera configuration and launch integration
- Common status and naming across supported ZED, RealSense, and OAK cameras

## Boundary

This package is provisional. Prefer upstream ROS camera drivers directly if no Kanga-specific management layer is needed.

This is an architecture placeholder; no 2026 implementation has been migrated yet.
