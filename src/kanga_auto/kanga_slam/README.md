# kanga_slam

RTAB-Map odometry, localisation, and SLAM integration for Kanga.

## Owns

- RTAB-Map odometry and SLAM launch configuration
- robot_localization configuration where used
- Sensor remapping and SLAM-related TF configuration

## Boundary

RTAB-Map remains an external dependency; do not copy its source into this repository.

This is an architecture placeholder; no 2026 implementation has been migrated yet.
